Pairs of labels and URLs, separated by a "|", each pair on a new line. Minimum one pair.
"%%u" (without the quotes) will be replaced by the URL that was clicked.
Everything before the "|" will be used as the context menu label.
The resulting full URL will be opened in a new tab. If you only need to invoke the GET request, consider the send-link extension.

Example:
Entry1|https://example.com?url=%%u
Entry2|https://example.com?url=%%u
